Protokoli in tehnike

Kratki izvlečki iz usposabljanj — za osnovno pripravljenost.
PROTOKOL // FILOZOFIJA

Realna samoobramba

Telo ne gre tja, kjer možgani še niso bili. Realna samoobramba je več kot borba — je preživetje.

PROTOKOL // NAČELA

Cilj ni zmaga, je preživetje

Na ulici ni pravil in ni sodnika. Cilj je prepoznati nevarnost, se ji izogniti in varno zapustiti situacijo.

PROTOKOL // MOTIVACIJA

Pripravljenost rešuje posameznika

Statistika tolaži množice. Treniraj danes, da lahko zaščitiš sebe, svoje in druge jutri.

PROTOKOL // OSVEŠČANJE

Nevarnost ne opozarja

Ne pride z napovedjo. Pride hitro in brez vprašanja — zato je pripravljenost stalna, ne priložnostna.

PROTOKOL // PRAVNI VIDIK

Kdo sme imeti solzivec

Polnoletne osebe brez posebnega dovoljenja — a uporaba je zakonita le v nujni in sorazmerni samoobrambi.

PROTOKOL // ZASNOVA PROSTORA

Varna pisarna, varni ljudje

Devet pravil za postavitev prostora — od nadzora izhoda do prepoznave opozorilnih znakov agresije.

PROTOKOL // NIVO OGROŽENOSTI

Rumena · Oranžna · Rdeča

Trije nivoji pripravljenosti glede na stopnjo nevarnosti — drža, glas in odziv za vsako fazo.

PROTOKOL // OCENA OSEBE

Prepoznavanje stanja

Zaznaj. Oceni. Odloči. Prepoznavanje jeze, strahu, panike in nevarnosti pri drugi osebi.

PROTOKOL // SKRIJ SE

Zabarikadiraj se

Ugasni naprave, pošlji sporočilo namesto klica, ne odpiraj vrat nikomur.

PROTOKOL // DIHANJE

Dihanje pod stresom

Vdih skozi nos 3–4 sekunde, tih izdih 5–6 sekund — tehnika, ki jo lahko uporabiš, ko si v nevarnosti.

PROTOKOL // PRVA POMOČ

Plinski razpršilec

Ukrepanje ob izpostavljenosti solzivcu — oči, dihala, koža in kdaj poklicati 112.

PROTOKOL // AMOK DOGODKI

Prepoznaj. Ukrepaj. Preživi.

Prepoznaj znake nevarnosti, ukrepaj hitro in pravilno (zbeži, skrij se, brani se) ter preživi s pripravljenostjo.
